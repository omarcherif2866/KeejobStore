Les tests de compétences

Le guide de l’équipe de recrutement sur les tests de compétences de pré-évaluation

Les tests de compétences peuvent être utilisés pour mesurer le rendement et l'efficience des employés. Les résultats de ces tests peuvent fournir aux gestionnaires une meilleure compréhension de la façon dont l'employé peut contribuer à l'atteinte des objectifs de l'organisation. Ces tests peuvent également être utilisés pour déterminer si des changements dans le travail ou dans les règles de travail sont nécessaires.
